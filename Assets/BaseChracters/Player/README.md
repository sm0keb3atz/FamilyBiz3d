# Player Character Assets

Active runtime assets live here:

- `FB_Player_Modular_v002.glb` and matching `FB_Player_Modular_v002_*` textures are the current base source/import files.
- `Meshes/Auto` contains modular runtime mesh resources used by `PlayerAppearanceComponent`.
- `Meshes/Police` contains fixed police uniform mesh resources used by `PlayerVisualModular.tscn`.
- `Materials/Modular` contains the material resources used by the appearance component.
- `Materials/Variants` contains optional auto-discovered material variants.
- `Textures` contains shared body/clothing texture maps referenced by materials.
